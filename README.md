## 📌 Project Description

This project presents a complete, industry-standard, end-to-end data analytics workflow
built on real customer shopping behavior data. The objective was to transform raw 
transactional data into meaningful business intelligence by applying professional 
data analytics techniques across Python, SQL, and Power BI.

The analysis covers customer spending patterns, product preferences, discount behavior, 
customer segmentation, and subscription trends — ultimately delivering actionable 
recommendations to support strategic business decisions.

---

## ✅ Completed Work

- ✔️ Data cleaning and preparation using Python (Pandas)
- ✔️ Exploratory Data Analysis (EDA) with statistical summaries
- ✔️ Feature engineering (age groups, purchase frequency)
- ✔️ Missing value handling using category-wise median imputation
- ✔️ MySQL database integration using SQLAlchemy
- ✔️ 10 structured SQL business queries for insight extraction
- ✔️ Interactive Power BI dashboard with dynamic filters
- ✔️ Business recommendations based on data findings
- ✔️ Professional project report and business problem statement

## 🚀 How to Use This Project

**1. Clone the repository**

- git clone https://github.com/HansikaDulanjani/Customer-Behavior-Data-Analyst-Portfolio-Project.git
- cd Customer-Behavior-Data-Analyst-Portfolio-Project

**2. Run the Jupyter Notebook**
- Open `Customer_Shopping_Behavior_Analysis.ipynb`
- Run all cells for data cleaning, EDA and MySQL connection

**3. Set up MySQL Database**
- Open MySQL Workbench
- Create a database: `CREATE DATABASE customer_trends;`
- The notebook will automatically load data into MySQL

**4. Run SQL Queries**
- Open `customer_behavior_sql_queries.sql` in MySQL Workbench
- Run queries to explore business insights

**5. Explore the Power BI Dashboard**
- Open `Customer_Behavior_Dashboard.pbix` in Power BI Desktop
- Connect to your MySQL database
- Interact with filters and visuals

## 💡 Key Business Insights

- Male customers generate significantly higher revenue than female customers
- Young Adults contribute the highest total revenue across all age groups
- Discount usage is highest for Hats, Sneakers and Coats
- Subscribed customers maintain a comparable average spend to non-subscribers
- Majority of customers (3,116) fall into the Loyal segment
